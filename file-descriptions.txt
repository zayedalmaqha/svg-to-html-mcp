SVG to HTML MCP 도구 파일 설명

이 문서는 SVG to HTML MCP 도구의 각 파일과 그 기능을 설명합니다.

■ 핵심 파일

src/index.ts
TypeScript로 작성된 MCP 서버의 소스 코드입니다. 다음 기능이 포함되어 있습니다:
- MCP 도구 정의 및 스키마
- TypeScript 코드를 HTML로 변환하는 로직
- 요청 처리 및 응답 생성
- 오류 처리 및 검증

dist/index.js
TypeScript 코드가 컴파일된 JavaScript 파일로, 실제 실행되는 파일입니다:
- npx 명령을 통해 GitHub에서 직접 실행할 때 진입점
- 모든 MCP 서버 로직 포함
- 배포용 파일

package.json
Node.js 프로젝트의 메타데이터와 의존성을 정의하는 파일입니다:
- 프로젝트 이름, 버전, 설명 등의 정보
- @modelcontextprotocol/sdk 등의 의존성
- 스크립트 명령어(build, prepare, start 등)
- bin 필드가 npx 실행을 위한 진입점(dist/index.js) 지정
- 엔진 요구사항 및 기타 설정

tsconfig.json
TypeScript 컴파일러 설정 파일로, 다음을 정의합니다:
- JavaScript 변환 대상 버전
- 모듈 시스템 설정
- 엄격한 타입 검사 옵션
- 출력 디렉토리 및 기타 컴파일러 옵션

■ 설치 및 실행 파일

install.js
Claude 데스크톱 앱 설정을 자동으로 업데이트하는 스크립트입니다:
- 운영체제 감지(Windows/macOS)
- TypeScript 코드 빌드 자동화
- Claude 설정 파일 위치 확인
- GitHub 저장소 방식으로 도구 설정 추가
- 오류 시 수동 설치 안내 제공

run.bat (선택 사항)
Windows에서 도구를 로컬에서 테스트하기 위한 배치 스크립트입니다.

run.sh (선택 사항)
macOS 및 Linux에서 도구를 로컬에서 테스트하기 위한 셸 스크립트입니다.

■ 설정 파일

.npmignore
npm 패키지 배포 시 포함하지 않을 파일을 정의합니다:
- 소스 코드(src/)
- 개발 환경 파일
- 테스트 파일
- 기타 불필요한 파일들

.gitignore
Git 버전 관리에서 제외할 파일과 디렉토리를 정의하는 파일입니다:
- node_modules/
- dist/
- 로그 파일
- 시스템 및 IDE 설정 파일

■ 문서 파일

README.txt
프로젝트의 메인 문서로, 다음 정보를 제공합니다:
- 도구 설명 및 기능
- 설치 방법
- 사용 방법
- 프로젝트 구조
- 문제해결 팁
- 라이선스 정보

README_FOR_NEWUSER.txt
새 사용자를 위한 더 자세한 설치 및 사용 가이드입니다.

FILE_DESCRIPTIONS.txt
현재 문서로, 저장소 내 각 파일의 역할과 기능을 설명합니다.
